# PHP Object-Oriented Programming (OOP) Assignment for Interns

Welcome! This guide will help you understand the concepts of Object-Oriented Programming (OOP) and how to implement them in PHP with an interactive HTML interface.

**Important Note:**
For the best interest of trainees to gain the most from the internship program.
Using AI tools (e.g., ChatGPT, GitHub Copilot) is strictly limited. Consult your trainer first before using any AI assistance to ensure proper learning and compliance.

---

## 📂 Assignments

1. **OOP Concepts Summary**
   - Review and check the provided documents related to Object-Oriented Programming (OOP).
   - Create a markdown file named `oop_summary.md` to summarize your understanding regarding OOP concepts (such as classes, objects, inheritance, polymorphism, encapsulation, and abstraction).

2. **PHP OOP Implementation with HTML UI**
   - Apply the OOP concepts you have learned to write PHP code.
   - Create an HTML User Interface (UI) that allows users to interact with your PHP objects. You can mix the HTML and PHP code in the same files for this exercise. Refer to tasks_1.md for more details.

By completing these assignments, you should be able to:
