# PHP Basic Programming Assignments for Interns

Welcome! This guide will help you get comfortable with PHP basics. By the end, you'll understand fundamental PHP syntax, control structures, and how to visualize control flow with flowcharts.

**Important Note:**
For the best interest of trainees to gain the most from the internship program.
Using AI tools (e.g., ChatGPT, GitHub Copilot) is strictly limited. Consult your trainer first before using any AI assistance to ensure proper learning and compliance.

---

## 📂 Assignments

1. **Project Structure**
   - Identify the `README.md` and other documentation files.
   - For each assignment in `task_<number>.md` file, understand the requirements and expected output. Create a new folder `task_<number>` containing PHP files for each task (e.g., `ex_1.php`, `ex_2.php`, etc.) and JPG files for the flowcharts (e.g., `ex_1.jpg`, `ex_2.jpg`, etc.) to implement the required functionality.

2. **Basic PHP Programming**
   - Implement a PHP script that performs given tasks.
   - Learn and practice basic syntax: variables, data types, operators, and echo/print statements.

3. **Control Structures**
   - Implement conditional statements: if, else, elseif, switch.
   - Use loops: for, while, do-while, foreach.
   - Practice nested control structures and logical operators.

4. **Flow Charts of Control Flow**
   - For each control structure, draw a flowchart illustrating the logic flow.
   - Use tools like draw.io to create diagrams.
   - Include examples for decision points, loops, and sequences.

5. **Integration and Testing**
   - Combine basic programming and control structures in a single script.
   - Test your code in a browser or command line.
   - Debug any errors and document your solutions.
