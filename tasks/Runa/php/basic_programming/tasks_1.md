# PHP Basic Programming Tasks

This file contains tasks adapted for 8 hours of practice. Focus on implementing mathematical series calculations using loops and basic PHP.

## Task List

### Exercise 1: Calculate S(n) = 1 + 2 + 3 + … + n (30 minutes)

- Write a PHP script to calculate the sum of the first n natural numbers.
- Use a for loop.
- Input n from user.

### Exercise 2: Calculate S(n) = 1^2 + 2^2 + … + n^2 (30 minutes)

- Calculate the sum of squares from 1 to n.
- Use a for loop.

### Exercise 3: Calculate S(n) = 1 + ½ + 1/3 + … + 1/n (30 minutes)

- Calculate the harmonic series sum up to n.
- Use a for loop with floating point division.

### Exercise 4: Calculate S(n) = ½ + ¼ + … + 1/2n (30 minutes)

- Sum of 1/(2k) for k=1 to n.
- Use a for loop.

### Exercise 5: Calculate S(n) = 1 + 1/3 + 1/5 + … + 1/(2n + 1) (45 minutes)

- Sum of odd denominators.
- Use a for loop.

### Exercise 6: Calculate S(n) = 1/1×2 + 1/2×3 +…+ 1/n x (n + 1) (45 minutes)

- Sum of 1/(k\*(k+1)) for k=1 to n.
- Use a for loop.

### Exercise 7: Calculate S(n) = ½ + 2/3 + ¾ + …. + n / n + 1 (45 minutes)

- Sum of k/(k+1) for k=1 to n.
- Use a for loop.

### Exercise 8: Calculate S(n) = ½ + ¾ + 5/6 + … + 2n + 1/ 2n + 2 (45 minutes)

- Sum of (2k+1)/(2k+2) for k=0 to n-1.
- Use a for loop.

### Exercise 9: Calculate T(n) = 1 x 2 x 3…x N (30 minutes)

- Calculate the factorial of n.
- Use a for loop.

### Exercise 10: Calculate T(x, n) = x^n (30 minutes)

- Calculate x raised to the power n.
- Use a for loop (no built-in pow).

### Exercise 11: Calculate S(n) = 1 + 1.2 + 1.2.3 + … + 1.2.3….N (1 hour)

- Sum of factorials up to n.
- Use nested loops or factorial function.
- Draw a flowchart for the control structure using draw.io.

### Exercise 12: Calculate S(n) = x + x^2 + x^3 + … + x^n (45 minutes)

- Geometric series sum.
- Use a for loop.

### Exercise 13: Calculate S(n) = x^2 + x^4 + … + x^2n (45 minutes)

- Sum of even powers.
- Use a for loop.

### Exercise 14: Calculate S(n) = x + x^3 + x^5 + … + x^2n + 1 (45 minutes)

- Sum of odd powers.
- Use a for loop.

### Exercise 15: Calculate S(n) = 1 + 1/1 + 2 + 1/ 1 + 2 + 3 + ….. + 1/ 1 + 2 + 3 + …. + N (1 hour)

- Complex sum involving cumulative sums.
- Use nested loops.
- Draw a flowchart for the control structure using draw.io.

## Notes

- Total estimated time: ~8 hours.
- Test all scripts with different values of n and x.
- For flowcharts, export as JPG and include in your submission.
- Focus on loop implementation and mathematical accuracy.
